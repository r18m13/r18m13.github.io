Collaborative Drone-UGV System for Precision Agriculture

A collaborative drone‑UGV system enabling efficient, autonomous precision agriculture with real‑time communication, coordinated operations, and targeted intervention.

Overview

The collaborative drone‑UGV system was developed to address the challenge of performing precision agricultural tasks efficiently and autonomously. Traditional methods of crop monitoring and intervention are labor-intensive, time-consuming, and prone to inefficiency. By enabling drones to detect crop conditions and communicate positional data to UGVs equipped with robotic manipulators, the goal was to create a coordinated system that is accurate, responsive, and minimizes human effort.

The solution integrates real-time aerial sensing with ground-based robotic action. Drones capture high-resolution imagery to identify target areas, while a communication pipeline transmits precise positional data to the UGV. The UGV’s manipulator executes targeted interventions such as weeding or planting, guided by the drone’s input. The system is engineered for synchronized operation, low-latency communication, and reliable field performance, resulting in an effective, autonomous precision agriculture workflow.

Technical Implementation

Architecture

The collaborative Drone‑UGV system integrates aerial weed detection with autonomous ground intervention. The drone is equipped with a Sony Alpha camera and NVIDIA Jetson Orin Nano for real-time weed detection and Ground Sampling Distance (GSD) calculation using GPS data via MAVLink. Detected weed coordinates are transmitted to the UGV for precise action.

The UGV, a Clearpath Husky A200, is equipped with a ZED stereo camera, 3D LiDAR, and a UR5e manipulator with a Robotiq gripper. A custom path-planning algorithm navigates the UGV to target GPS locations, applying positional offsets for accurate manipulator alignment. ROS handles localization, navigation, and sensor integration, while TCP ensures reliable communication between UGV and manipulator.

Example Python Socket Client for Sending Weed Coordinates to UGV:

import socket

HOST = '192.168.1.10' # UGV IP address
PORT = 5005

s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
s.connect((HOST, PORT))

weed_gps = "34.12345,-118.12345"
s.sendall(weed_gps.encode())
s.close()

Example URScript Command to Move UR5e to Target Pose:

movej([0.0, -1.57, 1.57, 0.0, 1.57, 0.0], a=1.2, v=0.25)
# a is acceleration, v is velocity
movel(p[0.5, 0.0, 0.2, 0, 3.14, 0], a=0.5, v=0.1)
# Move linearly to precise position with offset for weed uprooting

The system follows a modular, bidirectional communication architecture connecting the drone, UGV, and manipulator:

  • Drone Layer: Captures high-resolution imagery, runs onboard inference on Jetson Orin Nano, calculates GSD, and sends GPS coordinates of detected weeds.
  • UGV Layer: Clearpath Husky A200 receives GPS targets, localizes using ZED stereo camera, 3D LiDAR, and ROS, and navigates with custom path planning.
  • Manipulator Layer: UR5e with Robotiq gripper executes precise interventions using positional offsets communicated over TCP.
  • ROS Core: Manages navigation, manipulator control, and sensor fusion for reliable operation.
  • Monitoring Module: Tracks latency, dropped messages, and system performance to maintain responsiveness.

The modular design allows seamless coordination between aerial detection, ground navigation, and manipulator execution for autonomous precision agriculture.

Performance Optimizations

Several optimizations were implemented to achieve smooth, autonomous operation:

  • Onboard Inference Efficiency: Jetson Orin Nano processes weed detection in real-time, minimizing latency.
  • GPS to Manipulator Mapping: Positional offsets ensure UR5e gripper aligns accurately with target weeds.
  • Path Planning Optimization: Custom algorithm computes obstacle-free routes using LiDAR and stereo vision data.
  • Low-Latency Communication: TCP links between UGV and manipulator, and drone-to-UGV data transfer maintain synchronization.
  • Sensor Fusion: ROS integrates GPS, LiDAR, and stereo vision for robust localization and navigation.

These optimizations collectively reduce latency, enhance intervention precision, and ensure reliable autonomous operation in agricultural fields.

Results & Impact

The system provides a fully autonomous precision agriculture workflow: drones detect weeds, compute GPS locations with GSD, and transmit them to the UGV. The Husky A200 navigates autonomously to each target, and the UR5e manipulator with Robotiq gripper executes precise weed removal.

Low-latency communication ensures synchronized operation between aerial and ground units. Onboard Jetson inference reduces dependency on ground processing, while ROS-based sensor fusion provides robust GPS localization even in complex terrains.

The modular architecture supports rapid experimentation with new detection algorithms, manipulator strategies, and field layouts. Monitoring tools maintain reliability by tracking latency, message integrity, and system performance. Overall, the system demonstrates an effective, coordinated, and autonomous solution for precision agriculture.
